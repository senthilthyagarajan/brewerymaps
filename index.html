Breweries Near You !!

  •  About the project

A simple web app , mapping approximately 5600 breweries in US. The breweries data is from Brewers Association. The final data GeoJson and the R script to scrape ,clean and geocode can be found in my github project repo brewerymaps.

Features
  • Seach functionality using autocomplete.
  • Find 5 nearest breweries from your current location.
  • Filter the breweries that are being displayed in your current extent.
